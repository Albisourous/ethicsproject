WINDOWS 10:

Protect my user information

Source: https://www.privateinternetaccess.com/blog/how-to-disable-tracking-features-in-windows-10/


Cortana:

Press the Cortana button in the Windows taskbar.

Select the gear icon and enter the Settings.

Find the Cortana option that will give you suggestions, ideas, etc. Toggle that option off.


Targeted Advertising:

Go to the Privacy setting within the system settings.

Go to the General section and under Change Privacy Options. Visit the “Let apps use my advertising ID for experiences across apps”. Turn this option off.


Network Sharing:

Visit the Network and Internet settings in Windows 10.

Select Wi-Fi and then select Manage Wi-Fi Settings. Look at the Wi-Fi Sense option. Turn off the options: “Connect to Suggested Open Hotspots” “ Connect to Networks shared by my contacts”


File sharing:

Visit the Settings. Select Windows Update under Update and Security setting.

Select Advanced Options and go to “Choose how updates are delivered”.

Turn off the setting that says “Updates from more than one place”.
